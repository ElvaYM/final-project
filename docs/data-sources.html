Chapter 2 Data sources

Our data come from the GHTorrent Project, which is “an effort to create a scalable, queriable, offline mirror of data offered through the Github REST API.” It includes the complete data up to 2019-06-30 (in the format of MySql dumps and MongoDB daily dumps). Due to our limited computing power, we choose to use only one day’s data (the raw data is about 23 gigabytes), 2019-06-30.

The raw GHTorrent data consists of 16 tables and is stored in bson format. The schema is provided under the resources/schema folder:

##  [1] "commit_comments.txt"       "commits.txt"               "events.txt"                "forks.txt"                 "geo_cache.txt"            
##  [6] "issue_comments.txt"        "issue_events.txt"          "issues.txt"                "org_members.txt"           "pull_request_comments.txt"
## [11] "pull_requests.txt"         "repo_labels.txt"           "repos.txt"                 "topics.txt"                "users.txt"                
## [16] "watchers.txt"

And a typical schema file of the table includes three elements: the number of records, the schema of one record, and an example record:
